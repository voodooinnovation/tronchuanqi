Distribution pack:
------------------

The zip file has subdirectories in it. You must use the pkunzip -d option
to restore this directory tree or you will have problems because the files
will not be in their proper subdirectories.

This is the subdirectory layout:

.\                 Info directory
.\apps             Applications written using this component
.\demos            Sample applications
.\dll              MSIC.DLL library for using MSI reports/data from any app
.\reportviewer     Standalone Report Viewer for XML reports generated by MSI
.\journalviewer    Standalone Journal Viewer for journal files (*.mjf)
.\int              interface files (not in registered version)
.\D5               compiled units containing components and packages for installing for Delphi 5 (not in registered version)
.\D6               compiled units containing components and packages for installing for Delphi 6 (not in registered version)
.\D7               compiled units containing components and packages for installing for Delphi 7 (not in registered version)
.\Sources          source files (only in registered version)


Installation:
-------------
DELPHI 5/6/7: Using Delphi, do a file/open, select *.dpk and browse to
the directory D5/D6, choose MSI_D5_Rtl.dpk/MSI_D6_Rtl.dpk/MSI_D7_Rtl.dpk which is a runtime package source for all components. 
Click compile button. Then open MSI_D5_Dsgn.dpk/MSI_D6_Dsgn.dpk/MSI_D7_Dsgn.dpk and then click on the Install button. 
You should see the MiTeC tab on the component gallery.

!!! Be sure to uncheck 'Stop On Delphi Exception' option in Debugger Options|Language Exceptions.
